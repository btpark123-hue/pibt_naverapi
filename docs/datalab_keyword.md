# 검색어트렌드 - 네이버 데이터랩

## 개요

네이버 데이터랩의 **검색어트렌드**는 사용자가 직접 주제어와 그에 포함될 세부 검색어들을 설정하여 네이버에서 해당 키워드들의 검색 추이를 분석할 수 있는 서비스입니다.

단순히 특정 단어 하나만 보는 것이 아니라, 유사한 의미를 가진 여러 단어를 하나의 '주제어'로 묶어 통합적인 트렌드를 파악할 수 있는 것이 가장 큰 특징입니다.

## 주요 기능

### 1. 주제어 및 하위 검색어 설정

- **주제어**: 분석하고자 하는 큰 카테고리 명칭입니다. (예: 캠핑)
- **하위 검색어**: 해당 주제어에 포함될 구체적인 검색어들을 콤마(,)로 구분하여 입력합니다. (예: 캠핑, Camping, 캠핑용품, 텐트, 글램핑 등)
- 입력한 모든 하위 검색어의 검색량을 합산하여 주제어의 전체 트렌드를 보여줍니다.

### 2. 다각도 필터링 및 분석

- **기간 설정**: 일간, 주간, 월간 단위로 조회가 가능하며, 2016년부터 현재까지의 데이터를 제공합니다.
- **기기별 분석**: PC와 모바일 환경에서의 검색 비중을 나누어 볼 수 있습니다.
- **성별 분석**: 남성과 여성의 검색 트렌드 차이를 비교할 수 있습니다.
- **연령별 분석**: 0세부터 60세 이상까지 총 11개 구간의 연령별 데이터를 제공합니다.

### 3. 데이터 수치 (Ratio)

- 데이터랩에서 제공하는 수치는 절대적인 검색 건수가 아니라, 조회 기간 내에서 가장 많이 검색된 시점을 **100**으로 설정한 **상대적인 수치**입니다.
- 이를 통해 특정 시점 대비 현재의 관심도가 얼마나 높아졌는지, 혹은 낮아졌는지를 직관적으로 파악할 수 있습니다.

## 활용 가이드

### 효율적인 키워드 구성 팁

주제어 설정을 할 때 검색 사용자들이 혼용해서 사용하는 단어들을 모두 포함하는 것이 정확한 트렌드 파악에 도움이 됩니다.

- **예시 (반려동물)**: 주제어 '강아지' -> 하위어: 강아지, 애견, 반려견, puppy, 개
- **예시 (이커머스)**: 주제어 '아이폰' -> 하위어: 아이폰, iPhone, 아이폰15, 아이폰프로

### 비즈니스 활용 사례

1. **신제품 런칭 시점 결정**: 매년 반복되는 계절성 가전(에어컨, 히터 등)의 검색량이 급증하는 시점을 파악하여 마케팅 집중 기간을 설정할 수 있습니다.
2. **시장 트렌드 분석**: 경쟁 브랜드와 자사 브랜드의 주제어 검색량을 비교하여 시장 점유율의 변화 추이를 추측할 수 있습니다.
3. **콘텐츠 제작 키워드 선정**: 블로그나 유튜브 콘텐츠 제작 시 현재 급상승 중인 키워드를 파악하여 가시성을 높일 수 있습니다.

## API 연동 안내

네이버 데이터랩은 웹 서비스뿐만 아니라 개발자를 위한 **통합 검색어 트렌드 API**를 제공합니다. 이를 통해 프로그램 내에서 직접 트렌드 데이터를 수집하고 분석할 수 있습니다.

- **API 문서 보기**: [통합 검색어 트렌드 API 가이드](file:///c:/naver_api/docs/datalab_search_trend.md)
- **제공 데이터**: 주제어별 검색 상대 수치 (JSON 형식)
- **호출 한도**: 일 1,000회

## 주의사항

- 검색어에 특수문자가 포함된 경우 정확한 결과가 나오지 않을 수 있습니다.
- 개인정보 보호를 위해 검색 발생량이 매우 적은 경우에는 데이터가 0으로 노출될 수 있습니다.
- 로컬 파일 경로의 URL을 통한 공유 기능은 작동하지 않으므로 실제 웹 주소를 활용해야 합니다.

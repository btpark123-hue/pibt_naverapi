# 네이버 오픈메인 API - OpenMain API

## 개요

네이버 오픈메인 API는 여러분의 서비스나 컨텐츠를 네이버 모바일 메인의 'MY구독' 또는 기타 개인화 영역에 추가할 수 있도록 연결해주는 기능을 제공합니다.

사용자가 '네이버 메인에 추가' 버튼을 클릭하면, 여러분의 서비스가 네이버 메인의 홈 탭이나 전용 판에 등록되어 지속적인 방문을 유도할 수 있습니다.

## 주요 특징

- **접근성 향상**: 사용자가 네이버 앱을 열 때마다 여러분의 서비스에 쉽게 접근할 수 있는 경로를 제공합니다.
- **간편한 설정**: 복잡한 SDK 설치 없이 특정 URL 규격만 맞추면 바로 적용 가능합니다.
- **사용자 유지**: 고정적인 노출 지점을 확보함으로써 모바일 웹 서비스의 재방문율을 높일 수 있습니다.

## 구현 방법

### 1. 네이버 메인 추가 링크 생성

네이버 메인에 추가하기 위한 고유 스킴(Scheme) URL을 버튼이나 링크에 연결합니다.

- **기본 URL**: `https://naver.me/main/add` (또는 네이버에서 제공하는 특정 도메인)
- **요청 파라미터**:

  | 매개변수 | 타입 | 필수 | 설명 |
  | :--- | :--- | :--- | :--- |
  | `url` | String | Y | 네이버 메인에 등록할 서비스의 URL (UTF-8 인코딩) |
  | `title` | String | Y | 네이버 메인에서 보여질 서비스 명칭 |

### 2. 버튼 디자인 가이드

네이버에서는 사용자에게 익숙한 '네이버 메인에 추가' 전용 버튼 이미지를 제공합니다. 가이드에 맞는 버튼을 사용하여 신뢰도를 높이는 것이 좋습니다.

#### 공식 버튼 예시 (HTML)

```html
<a href="https://naver.me/main/add?url=https%3A%2F%2Fmyservice.com&title=%EB%82%B4%EC%84%9C%EB%B9%84%EC%8A%A4">
  <img src="https://ssl.pstatic.net/static/share/image/btn_opening.png" alt="네이버 메인에 추가">
</a>
```

## 상세 구현 시나리오

### 시나리오 A: 모바일 웹 서비스 등록

사용자가 모바일 브라우저로 접속했을 때, "네이버 앱에서도 편하게 보세요"라는 문구와 함께 버튼을 노출합니다.

```javascript
function addToNaverMain() {
    var serviceUrl = encodeURIComponent("https://myweb.com/home");
    var serviceTitle = encodeURIComponent("데일리 뉴스레터");
    var addUrl = "https://naver.me/main/add?url=" + serviceUrl + "&title=" + serviceTitle;
    
    // 네이버 앱 내부 브라우저인 경우와 외부 브라우저인 경우를 체크하여 처리
    location.href = addUrl;
}
```

### 시나리오 B: 특정 판(Topic Card) 추가

특정 주제(경제, 스포츠 등)를 다루는 서비스인 경우, 해당 주제와 밀접한 네이버 판에 바로 추가되도록 유도할 수 있습니다. (네이버와의 별도 제휴가 필요할 수 있음)

## 기술적 제약 사항

1. **URL 유효성**: 등록할 URL은 네이버 서버에서 접근 가능한 공개된 URL이어야 합니다.
2. **도메인 제한**: 스팸 방지를 위해 네이버 내부 검수 시스템에 의해 특정 도메인이 차단되거나 등록이 제한될 수 있습니다.
3. **사용자 환경**: 주로 네이버 모바일 앱 환경에서 최적의 동작을 보장하며, PC 브라우저에서는 환경에 따라 '네이버 로그인이 필요합니다' 등의 메시지가 노출될 수 있습니다.

## 마케팅 및 디자인 팁

- **아이콘 최적화**: 사용자가 네이버 메인에서 여러분의 서비스를 쉽게 찾을 수 있도록 고유한 파비콘(Favicon)이나 대표 이미지를 잘 설정해야 합니다.
- **문구 선정**: 단순히 '추가하기' 보다는 '매일 아침 8시, 최신 정보를 네이버에서 바로 확인하세요'와 같이 사용자 혜택을 강조하는 문구를 버튼 주위에 배치하세요.

## 이용 문의 및 지원

- 오픈메인 설정 도중 발생하는 오류는 [네이버 개발자 포럼](https://developers.naver.com/forum)을 통해 문의할 수 있습니다.
- 대규모 트래픽이 예상되는 제휴 서비스의 경우 네이버 서비스 담당자와의 협의가 권장됩니다.

## FAQ

- **Q: 버튼을 눌러도 반응이 없습니다.**
  - A: `url` 파라미터가 정확히 인코딩되었는지, 그리고 대중적인 브라우저 환경인지 확인하세요.
- **Q: 추가된 내용을 어떻게 삭제하나요?**
  - A: 사용자가 네이버 앱의 '메인 편집' 메뉴에서 직접 등록된 서비스를 삭제하거나 순서를 변경할 수 있습니다.
